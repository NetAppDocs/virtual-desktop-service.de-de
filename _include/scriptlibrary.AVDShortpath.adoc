== AVD-ShortPath aktivieren

Dieses Skript-Paket aktiviert/deaktiviert die native AVD-Funktion _RDP Shortpath_, indem Sie den (relevanten) Befehl mit PowerShell ausf√ºhren:

Aktivieren:

`reg add "HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /v fEnableScreenCaptureProtection /t REG_DWORD /d 1`

Deaktivieren:

`reg delete "HKLM\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" /v fEnableScreenCaptureProtection /f`
